mantidimaging.core.reconstruct.tomopy_reconstruction module¶

mantidimaging.core.reconstruct.tomopy_reconstruction.reconstruct(sample, cor=None, proj_angles=None, ncores=None, progress=None)[source]¶

Performs a volume reconstruction using sample data provided as sinograms.

Parameters

  • sample – Array of sinogram images

  • cor – Array of centre of rotation values

  • proj_angles – Array of projection angles

  • ncores – Number of CPU cores to execute on, defaults to all

  • progress – Optional progress reporter

Returns

3D image data for reconstructed volume
