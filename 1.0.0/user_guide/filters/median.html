Median¶

Applies SciPy’s median filter (scipy.ndimage.filters.median_filter) (docs).

Python API¶

mantidimaging.core.filters.median_filter.execute(data, size, mode='reflect', cores=None, chunksize=None, progress=None)[source]¶
Parameters

  • data – Input data as a 3D numpy.ndarray

  • size – Size of the kernel

  • mode – The mode with which to handle the endges. One of [reflect, constant, nearest, mirror, wrap].

  • cores – The number of cores that will be used to process the data.

  • chunksize – The number of chunks that each worker will receive.

Returns

Returns the processed data
