Cut Off¶

Reduces the maximum intensity of an image by clipping values above a threshold defined as a factor of the maximum intensity.

For example, if the pixel intensity in an image is 2.7 and the cut off threshold level is 0.95 then every pixel with an intensity value grater than 2.56 (2.7 \times 0.95) will be clipped to 2.56.

Python API¶

mantidimaging.core.filters.cut_off.execute(data, threshold, progress=None)[source]¶

Cut off values above threshold relative to the max pixels.

Parameters

  • data – Input data as a 3D numpy.ndarray

  • threshold – The threshold related to the minimum pixel value that will be clipped

Returns

The processed 3D numpy.ndarray
