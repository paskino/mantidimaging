mantidimaging.core.process_list.process_list module¶

class mantidimaging.core.process_list.process_list.ProcessList[source]¶

Bases: object

Stores a queue of functions to be executed in first-in first-out order.

Currently only works with MantidImaging internal functions.

store(func, *args, **kwargs)[source]¶

Store a function in the process list to be executed later.

The arguments must be literal values.

Note: More arguments can be appended when the funciton is called for execution.

Parameters

  • func – This works with a function reference, because metadata needs to be read from the function.

  • args – Arguments that will be forwarded to the function. They must be literal values.

  • kwargs – Keyword arguments that will be forwarded to the function. They must be literal values.
