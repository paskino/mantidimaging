mantidimaging.core.filters.rebin.rebin module¶

mantidimaging.core.filters.rebin.rebin.execute(data, rebin_param, mode, cores=None, chunksize=None, progress=None)[source]¶
Parameters

  • data – Sample data which is to be processed. Expected in radiograms

  • rebin_param – int, float or tuple int - Percentage of current size. float - Fraction of current size. tuple - Size of the output image (x, y).

  • mode – Interpolation to use for re-sizing (‘nearest’, ‘lanczos’, ‘bilinear’, ‘bicubic’ or ‘cubic’).

  • cores – The number of cores that will be used to process the data.

  • chunksize – The number of chunks that each worker will receive.

Returns

The processed 3D numpy.ndarray
