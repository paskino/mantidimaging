Outliers¶

Important

This filter requires TomoPy to be available.

Removes bright or dark spots (outliers) from an image (or stack of images).

This uses the TomoPy implementation (tomopy.misc.corr.remove_outlier) (docs).

Python API¶

mantidimaging.core.filters.outliers.execute(data, diff, radius=3, mode='bright', cores=None, progress=None)[source]¶

Requires tomopy to be available.

Parameters

  • data – Input data as a 3D numpy.ndarray

  • diff – Pixel value difference above which to crop bright pixels

  • radius – Size of the median filter to apply

  • mode – Spot brightness to remove. Either ‘bright’ or ‘dark’.

  • cores – The number of cores that will be used to process the data.

Returns

The processed 3D numpy.ndarray
