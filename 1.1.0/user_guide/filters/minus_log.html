Minus Log¶

Important

This filter requires TomoPy to be available.

This filter applies the -ln(x) operation to the intensity of each pixel in an image.

This uses the TomoPy implementation (tomopy.prep.normalize.minus_log) (docs).

This is used to convert from a transmission image (i.e. high intensity indicating transmission through the sample) to attenuation images (i.e. high intensity indicating attenuation by the sample).

Python API¶

mantidimaging.core.filters.minus_log.execute(data, minus_log=True, progress=None)[source]¶

This filter should be used on transmission images (background corrected images).

It converts the images from transmission to attenuation.

Parameters

  • data – Sample data which is to be processed. Expected in radiograms

  • minus_log – Specify whether to calculate minus log or just return.

Returns

Inverted image
