Circular Mask¶

Important

This filter requires TomoPy to be available.

Applies TomoPy’s circular mask filter (tomopy.misc.corr.circ_mask) (docs).

Takes a radius as a ratio of the smallest image dimension and a value to apply to all pixels outside this radius.

Python API¶

mantidimaging.core.filters.circular_mask.execute(data, circular_mask_ratio, circular_mask_value=0.0, cores=None, progress=None)[source]¶
Parameters

  • data – Input data as a 3D numpy.ndarray

  • circular_mask_ratio – The ratio to the full image. The ratio must be 0 < ratio < 1

  • circular_mask_value – The value that all pixels in the mask will be set to.

Returns

The processed 3D numpy.ndarray
