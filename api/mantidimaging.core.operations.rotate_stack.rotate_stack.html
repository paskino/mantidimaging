mantidimaging.core.operations.rotate_stack.rotate_stack module¶

class mantidimaging.core.operations.rotate_stack.rotate_stack.RotateFilter[source]¶

Bases: mantidimaging.core.operations.base_filter.BaseFilter

Rotates the image data by an arbitrary degree counter-clockwise.

Intended to be used on: Projections

When: Rarely/never, ASTRA vector geometry will take care of the tilt correction

Caution: Manually rotating could introduce additional artifacts in the reconstructed volume, and it is not strictly required as using vector geometry will correct for the tilt without manual rotation.
