mantidimaging.core.operations.remove_stripe_sorting_fitting.remove_stripe_sorting_fitting module¶

class mantidimaging.core.operations.remove_stripe_sorting_fitting.remove_stripe_sorting_fitting.RemoveStripeSortingFittingFilter[source]¶

Bases: mantidimaging.core.operations.base_filter.BaseFilter

Combination of algorithm 3 and 1 in [1]. Remove stripes using the sorting and fitting technique. Angular direction is along the axis 0.

Source: https://github.com/nghia-vo/sarepy

Intended to be used on: Sinograms

When: If stripes artifacts are present that have not been removed with outliers + flat-fielding the projections

Caution: Horizontal stripes are caused by changes in image intensity (pixel values), and should be fixed by ROI Normalisation instead!
