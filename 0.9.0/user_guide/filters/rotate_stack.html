Rotate Stack¶

Rotates an image (or each image in a stack) clockwise by a multiple of 90 degrees.

Through the Python API the flat and dark images can also be provided to process them in a single call to execute().

Command line examples¶

  • mantidimaging -i /some/data -r 1

  • mantidimaging -i /some/data --rotation 1

Python API¶

mantidimaging.core.filters.rotate_stack.execute(data, rotation, flat=None, dark=None, cores=None, chunksize=None, progress=None)[source]¶

Rotates a stack (sample, flat and dark images).

This function only works with square images.

If the picture is cropped first, the ROI coordinates have to be adjusted separately to be pointing at the NON ROTATED image!

Parameters

  • data – stack of sample images

  • rotation – The rotation to be performed

  • flat – flat images average

  • dark – dark images average

  • cores – cores for parallel execution

  • chunksize – chunk for each worker

Returns

rotated images
