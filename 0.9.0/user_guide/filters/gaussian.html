Gaussian¶

Applies SciPy’s Gaussian filter (scipy.ndimage.filters.gaussian_filter) (docs).

Command line examples¶

  • mantidimaging -i /some/data --gaussian-size 3

  • mantidimaging -i /some/data --gaussian-size 3 --gaussian-mode 'nearest'

  • mantidimaging -i /some/data --gaussian-size 3 --gaussian-mode 'nearest' --gaussian-order 1

Python API¶

mantidimaging.core.filters.gaussian.execute(data, size, mode, order, cores=None, chunksize=None)[source]¶
Parameters

  • data – Input data as a 3D numpy.ndarray

  • size – Size of the kernel

  • mode – The mode with which to handle the endges. One of [reflect, constant, nearest, mirror, wrap].

  • order – The order of the filter along each axis is given as a sequence of integers, or as a single number. An order of 0 corresponds to convolution with a Gaussian kernel. An order of 1, 2, or 3 corresponds to convolution with the first, second or third derivatives of a Gaussian. Higher order derivatives are not implemented

  • cores – The number of cores that will be used to process the data.

  • chunksize – The number of chunks that each worker will receive.

Returns

The processed 3D numpy.ndarray
