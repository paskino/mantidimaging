Full configuration (pre-proc + tool/algorithm + post-proc.

static empty_init()[source]¶

Create and return a ReconstructionConfig with all the default values.

This function is provided here to create a config with the defaults, but not go through the hassle of importing every single config and then constructing it manually. This method does that for you!
